# NezuWorksBot

## 概要

[line-works-sdk](https://github.com/nanato12/line-works-sdk)を使用して作成した、多機能BOTです。

### 使用中のSDKバージョン

- [line-works-sdk v3.4](https://github.com/nanato12/line-works-sdk/releases/tag/v3.4)

## 主な機能や特徴

### 実装済みのコマンドと説明

注意: 接頭辞はデフォルトで"!"です、

- `help`: ヘルプメッセージを送信
- `test`: テストメッセージを送信

### 技術スタック
- **SDK**: [line-works-sdk v3.4](https://github.com/nanato12/line-works-sdk/releases/tag/v3.4)
- **言語**: Python
- **主要ライブラリ**:
  - `line-works-sdk`
  - `python-dotenv`
  - `rich`

## プロジェクト詳細

### 目的
このBOTは、LINE WORKSプラットフォーム向けのカスタマイズ可能なボットアプリケーションです。SDKの更新に応じて継続的に改良・拡張されます。

### 倫理的ガイドライン
- LINE WORKSプラットフォームの規則を厳密に遵守
- プラットフォームの利用規約と指針を尊重

## セットアップ

### 必要な環境
- Python 3.9+
- pip
